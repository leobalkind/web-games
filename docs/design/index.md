# BORK BATTLE — Design Docs

The original meme-pug arena game.

## Overview

- **Name:** BORK BATTLE
- **Genre:** .io-style top-down arena, evolution-driven, single-player vs bots (multiplayer-ready architecture)
- **Audience:** teen / young adult meme crowd
- **Tone:** Cute but fierce
- **Pace:** Frantic, 1–3 minute matches
- **Hook:** Visible transformation — both pug AND vehicle dramatically evolve
- **Stack:** PixiJS + Vite, pixel art (32×32 base, 8-frame smooth, 64+ colors)

- [`characters.md`](./characters.md) — full 75-character roster across 5 tiers, with vehicles + sidekicks + abilities
