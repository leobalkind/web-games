# BORK BATTLE — Character Roster

**75 characters** organized into 5 tiers and themed groups. Each character has a unique pug + custom vehicle + sidekick passenger + signature ability. Designed for visual variety (silhouette, color, vibe) and mechanical variety (tank / glass cannon / support / trickster / brawler).

**Roster design principles:**
- Each tier = visibly bigger and more powerful than the last
- Within a tier, characters differ by SILHOUETTE first, color second, accessories third (so they read instantly even as 32×32 pixels)
- ~5–7 characters per theme group for cosmetic-set / collection appeal
- 6 military vehicles spread across tiers (per user spec: 5–8 military, rest meme/funny)
- A few "wild card" characters per tier that subvert the theme — these become fan favorites
